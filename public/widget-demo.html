Interactive demo of the embeddable chat widget

About This Demo

This page demonstrates the Tandym.ai chat widget in action. The widget appears as a floating button in the bottom-right corner of the page.

Click the chat button to open the widget and start chatting with the creator's AI twin!

How It Works

  1. The widget loads asynchronously and doesn't slow down your page
  2. It appears as a customizable floating button
  3. Clicking the button opens a chat interface
  4. Users can have conversations with your AI twin
  5. The widget is fully responsive and mobile-friendly
